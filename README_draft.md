# Praxis IQ

Institutional Intelligence Platform for Investor Relations.

## Purpose

Praxis IQ converts capital markets information into better management decisions.

## Initial Build Priorities

1. PostgreSQL data model
2. SEC EDGAR ownership ingestion
3. Peer ownership scoring
4. Investor recommendation engine
5. Mobile-first executive decision dashboard

## Core Product Principle

Precision over volume. One verified decision-maker is more valuable than fifty generic contacts.
